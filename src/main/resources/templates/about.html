About TravelGuard

We are a modern insurance provider dedicated to making travel protection simple, transparent, and accessible.

Development Details

This application is a demonstration of a full-stack web application built with modern technologies.

  • Backend: Java, Spring Boot, Spring Security, Spring Data JPA
  • Frontend: Thymeleaf, HTML5, CSS, Bootstrap 5, JavaScript
  • Database: H2 In-Memory Database
  • Architecture: Model-View-Controller (MVC)
